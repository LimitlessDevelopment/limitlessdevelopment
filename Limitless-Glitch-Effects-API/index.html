Limitless Glitch effects Documentation

Limitless Glitch Effects on Unity Asset Store

Unity provides 3 Render Pipelines with different capabilities. Limitless Glitch Effects supports all 3 render pipelines.

Inside "Assets\LimitlessUnityDevelopment\Limitless Glitch" folder you can find 3 unity packages.

1.“LimitlessGlitch_URP_Effects.unitypackage” for Universal Render Pipeline. 2. “LimitlessGlitch_HDRP_Effects.unitypackage” for High Definition Render Pipeline. 3. “LimitlessGlitch_StandardRender_Effects.unitypackage” for Built-In Render Pipeline.

Import package according to your Render Pipeline.

After importing package please follow the instructions for your Render Pipeline.

  • URP

  • HDRP

  • Built-In

Built-In

Setup NEW Effects fot Post processing stack v2.

  • To use new effects collection make sure that you installed "Post processing" package from package manager (Window/Package Manager) !

  • To setup Post processing stack v2 use this instructions https://docs.unity3d.com/Packages/com.unity.postprocessing@2.1/manual/Quick-start.html

  • In post process volume/profile click Add Effect/Limitless Glitch/"needed effect".

**Manipulating effects variables **

https://docs.unity3d.com/Packages/com.unity.postprocessing@2.0/manual/Manipulating-the-Stack.html

URP

Setup NEW Effects for Universal Render Pipeline.

  1. Open your Renderer(Forward Render by default) and click "Add Render Feature".
  2. Add all Limitless Glitch effects you need.
  3. Change Event in all effects to "Before Rendering Post-processing"
  4. For best look customize the order of Limitless Glitch effects.
  5. In your scene add Volume component to any gameobject you want. In volume component create new volume profile or use existing one. Click "override" and select needed effect.

Setup Example Scenes Effects:

  • In your UniversalRenderPipelineAsset (In "Asset/settings" folder by default) in Renderer List select "Limitless_Renderer".

Use "Global Post Processing Settings" to enable or disable Post Processing in scene view or via camera setup. THIS SETTING SHOULD BE TURNED OFF FOR EFFECTS, IN CASE OF USING THEM FOR SEPARATE LAYERS

HDRP

Setup NEW Effects for High Defenition Render Pipeline.

HDRP allows you to customize the order of your custom post-processing effect at each injection point. To order your effects:

1. Go to Edit > Project Settings and select the HDRP Default Settings tab.

2. Scroll down until you find the Custom Post Process Orders section. This section contains three lists, one for each injection point.

3. Add Retro Look Pro effects to these lists so that HDRP can render them.
